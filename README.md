
# Combine-playground
The app created as playground to investigate ability of the Combine
- timers
- notification center 
- combine latest 
- @Publishded 
- AnyPublisher

#### Outcome
Advantages
- Combine is good enough to replace old RXSwift code.
- Combine can be used for UIKit
- easy to read

Disadvantages 
- Developers need some time to learn new framework
